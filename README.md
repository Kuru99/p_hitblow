# 🎮 Hit & Blow

シンプルでありながら奥深い推理ゲーム「Hit & Blow（ヒット・アンド・ブロー）」です。
数字やアルファベットを用いた頭脳戦を、1人プレイ・ローカル対戦・Webブラウザを使ったオンライン通信対戦の様々なモードで楽しむことができます。
リンク:https://p-hitblow.onrender.com

## ✨ 主な機能

- **豊富なプレイモード**: 
  - **1人プレイ**: AIが考えた秘密のコードを当てるモード
  - **ローカル対戦**: 1台のPCで2人が交互に入力して対戦
  - **オンライン通信対戦 (WebSocket)**: 離れた友達とブラウザ上でリアルタイム対戦！
- **2種類のゲームシステム**:
  - **数字モード (0-9)**: 従来の数字当てゲーム
  - **英字モード (A-Z)**: アルファベットの組み合わせを当てる高難易度モード
- **オンライン対戦の充実した機能**:
  - ルーム作成・参加機能（4桁のIDで友達とマッチング）
  - 現在のサーバー接続人数のリアルタイム表示
  - 両プレイヤーの「準備OK」による待機・同期機能

## 🚀 インストール & 遊び方

このプロジェクトは [uv](https://github.com/astral-sh/uv) を使用してパッケージ管理されています。

### 1. リポジトリのクローンと環境構築
```bash
git clone https://github.com/Kuru99/p_hitblow.git
cd p_hitblow
uv sync
```

### 2. ゲームの起動
以下のコマンドでCLIの起動メニューが開きます。
```bash
uv run python -m hitblow
```

起動するとメニューが表示されます。
```
=== Hit & Blow ===
1: 1人プレイ
2: 対戦モード（ローカル）
3: 通信対戦（WebSocket）
プレイ人数を選んでください（1 / 2 / 3） > 
```
- **1, 2** を選ぶとターミナル上でそのままゲームが開始されます。
- **3** を選ぶとブラウザが立ち上がり、オンライン対戦のWebページに遷移します。

## 🌐 オンライン通信対戦について

Webアプリ版は [Render](https://render.com/) 等のPaaSでホスティングすることを前提としています。
ブラウザ版では以下の手順で対戦を行います：

1. **ホスト（部屋を作る人）** が「部屋を作成」し、モード（数字/英字）と桁数を選択。
2. 作成された **4桁の部屋ID** を対戦相手に伝える。
3. **ゲスト（参加する人）** はそのIDを入力して「部屋に参加」する。
4. 双方が **「準備OK」** を押すとゲーム開始！

## 🛠 開発者向け

テストの実行には `pytest` を使用します。
```bash
uv run pytest
```

### 構成
- `src/hitblow/core.py`: ゲームのコアロジック（Hit/Blow判定・正解生成）
- `src/hitblow/game.py` & `game_battle.py`: CLI向けのゲーム進行ループ
- `src/hitblow/web/app.py`: FastAPI と WebSocket を利用したバックエンドサーバー
- `src/hitblow/web/static/index.html`: ブラウザ用のフロントエンドUI
